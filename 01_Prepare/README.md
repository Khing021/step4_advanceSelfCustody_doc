# เตรียมตัวก่อนมาเรียนบันไดขั้นที่ 4: Advance Self-custody 🔐

กิจกรรมในคลาสวันที่ 26 เมษายน 2026 จะมีการใช้งานซอฟต์แวร์ที่หลากหลาย และทั้งหมดเกี่ยวข้องโดยตรงกับความปลอดภัย&ความเป็นส่วนตัวของเงินในกระเป๋าทุกท่าน

แนวปฏิบัติที่ดีที่สุด (Best Practiece) สำหรับการใช้งานซอฟต์แวร์เหล่านี้คือต้องตรวจสอบความถูกต้องของซอฟต์แวร์อย่างละเอียดก่อนที่จะติดตั้งลงในคอมพิวเตอร์และเริ่มต้นใช้งาน

แต่เพื่อความสะดวกรวดเร็วในการเรียนการสอน ผมจึงได้ทำคู่มือสำหรับการดาวน์โหลด ตรวจสอบ และติดตั้งซอฟต์แวร์เหล่านี้มาให้ทุกท่านได้เตรียมตัวกันก่อนล่วงหน้า โดยเฉพาะอย่างยิ่ง `Wasabi Wallet` ซึ่งจะต้องมีการซิงค์ข้อมูลให้เสร็จก่อนจึงจะเริ่มใช้งานได้ อาจกินเวลาหลายนาทีหรือหลายชั่วโมง จึงเป็นการดีหากทุกท่านติดตั้งซอฟต์แวร์ตัวนี้และซิงค์ข้อมูลที่จำเป็นให้เสร็จก่อนถึงวันเรียน

> [ ⚠️ คำเตือน ] การตรวจสอบซอฟต์แวร์เป็นขั้นตอนที่ยุ่งยาก แต่การดาวน์โหลด ติดตั้ง และใช้งานซอฟต์แวร์เหล่านี้เลยโดยไม่ตรวจสอบมีความเสี่ยง ซึ่งเราไม่แนะนำ และทักษะในการตรวจสอบความถูกต้องของซอฟต์แวร์เป็นทักษะที่สำคัญและมีประโยชน์ อยากให้ทุกคนได้ฝึกฝนไว้ให้เป็นนิสัย

## ข้อควรระวังพื้นฐาน
อย่าค้นหาไฟล์หรือหน้าเว็บไซต์ของซอฟต์แวร์เหล่านี้ด้วย serach engine (อย่างเช่น Google) ท่านอาจพบเว็บไซต์ปลอมที่สร้างขึ้นมาเพื่อหลอกลวง ให้ไปยังแหล่งต่าง ๆ ด้วยลิงก์ที่ให้ไว้ในคู่มือนี้เท่านั้น
