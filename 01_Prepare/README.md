# บันไดขั้นที่ 4: Advance Self-custody 🔐

คลังคู่มือการติดตั้งและใช้งาน Bitcoin Wallet สำหรับหลักสูตรการดูแลเหรียญด้วยตัวเองระดับก้าวหน้า (Advance Self-custody) โดยเน้นความปลอดภัยและการตรวจสอบความถูกต้องของซอฟต์แวร์เป็นสำคัญ

## 📁 เนื้อหาในชุดคู่มือ
1.  **[Sparrow Wallet](./sparrow_installation.md)** - คู่มือการติดตั้งและการตรวจสอบ PGP ของ Sparrow Wallet (เน้นการจัดการ UTXO และฟีเจอร์ระดับสูง)
2.  **[Wasabi Wallet](./wasabi_installation.md)** - คู่มือการติดตั้งและการใช้งาน Wasabi Wallet (เน้นความเป็นส่วนตัวและการทำ CoinJoin)
3.  **[Nunchuk Wallet](./nunchuk_installation.md)** - คู่มือการติดตั้ง Nunchuk Wallet (เน้นการทำ Multisig และการวางแผนมรดก)
4.  **[GPG Installation](./gpg_installation.md)** - วิธีการติดตั้งเครื่องมือ GPG (GNU Privacy Guard) สำหรับทุกระบบปฏิบัติการ

## ⚠️ ข้อควรระวังพื้นฐาน
- ดาวน์โหลดซอฟต์แวร์จากแหล่งที่มาทางการเท่านั้น
- ตรวจสอบลายเซ็นดิจิทัล (Verify Signature) ทุกครั้งก่อนติดตั้ง
- ห้ามเก็บ Seed Phrase หรือ Passphrase ไว้ในรูปแบบออนไลน์หรือถ่ายรูปเก็บไว้
