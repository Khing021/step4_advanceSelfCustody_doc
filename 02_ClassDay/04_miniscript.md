# Miniscript

## 0. เปลี่ยนเป็น Testnet

เพื่อความสบายใจในกิจกรรมนี้เราจะใช้งาน Bitcoin Testnet แทนที่จะเป็นบิตคอยน์จริง

- ใน Nunckuk Desktop ให้กดที่ปุ่มวงกลมมุมซ้ายล่างของหน้าจอแล้วเลือก `Setting` > `Network Setting` > กดเลือกหน้าช่อง `TESTNET SERVER`
- ใน Nunchuk Mobile ให้กดที่ `Profile` มุมขวาล่างของหน้าจอแล้วเลือก `Network Setting` > กดเลือกหลังช่อง `TESTNET SERVER`

## 1. สร้าง Group wallet

คนหนึ่งในกลุ่มรับหน้าที่สร้าง Group wallet > เลือกประเภทกระเป๋าเป็น Miniscript แบบ `Decaying Multisig`

- กรณีกลุ่ม 2 คน สร้างกระเป๋าแบบ 2-of-2 แต่หากเลยเวลา 16:00 น. ของวันนี้ (26 เมษายน 2026) จะลดจำนวนคีย์ที่่ต้องใช้ลงเหลือเป็น 1-of-2
- กรณีกลุ่ม 3 คน สร้างกระเป๋าแบบ 3-of-3 แต่หากเลยเวลา 16:00 น. ของวันนี้ (26 เมษายน 2026) จะลดจำนวนคีย์ที่่ต้องใช้ลงเหลือเป็น 1-of-3

เมื่อตั้งค่ากระเป๋าเรียบร้อยแล้ว ให้เพื่อนคนอื่นในกลุ่มช่วยกันเพิ่ม key ของตัวเองเข้าไป

## 2. ส่ง Address

ตัวแทนกลุ่มกด receive ใน wallet ที่สร้างขึ้นใหม่นั้นแล้วนำ address ที่ได้มากรอกใน [Google Form นี้](https://forms.gle/9nECddx8qV2bdcLq8)

(address ประเภท Native Segwit ใน Testnet จะขึ้นต้นด้วย `tb1q...`)

## 3. ครูส่งเงินให้

ครูจะส่งเงินจำนวนหนึ่งเข้าไปในกระเป๋า Multisig ของแต่ละกลุ่มโดยอ้างอิงจาก address ที่ส่งมา

## 4. ส่งเงินคืนให้ครู

ให้ทุกกลุ่มส่งเงินคืนให้ครู โดยใช้ address ของครูที่ให้ไว้
